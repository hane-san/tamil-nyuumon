動詞の基本活用イメージ：போ（行く） ♪

人称現在過去未来
நான்nāṉナーンபோறேன்போனேன்போவேன்
நீnīニーபோறேபோனேபோவே
நீங்கnīṅgaニーンガபோறீங்கபோனீங்கபோவீங்க
அவன்avaṉアヴァンபோறான்போனான்போவான்
அவள்avaḷアヴァルபோறாபோனாபோவா
அவங்கavaṅgaアヴァンガபோறாங்கபோனாங்கபோவாங்க

この表は「போ（行く）」だけを使った入口です。まず、語幹そのものよりも、語尾が人称と時制を同時に持つことを見ます。会話では நீங்க や அவங்க のような口語形がかなり使いやすいです。

名詞の格変化ルール

  • -am 終わり：மரம் → மரத்து（木 → 木に格語尾を付ける語幹）
  • -ṭu 終わり：வீடு → வீட்டு（家 → 家に格語尾を付ける語幹）
  • -ṟu 終わり：ஆறு → ஆற்று（川 → 川に格語尾を付ける語幹）
  • ā 終わりの名前：சீதா → சீதாவை / சீதாவுக்கு のように வ் が入りやすいです。
  • 初心者の入口では、まず「語幹にしてから格語尾が付く」と見るのが安全です。

カタカナ発音ルール

  • 語頭 e / ē は、読み上げ補助では「イェ / イェー」に寄せる。
  • k, t, p は、母音にはさまれると「ガ・ドゥ・バ」寄り。重なると「ッ」で出す。
  • c が重なると「ッチャ」寄りにする。
  • ṭ は母音間で「ドゥ」寄り、重なると「ットゥ」。
  • ṟ は単独ならラ行寄り、重なる ṟṟ は「ットゥ」寄りにする。
  • カタカナ欄は読みやすさ優先。細かい区別はタミル文字とローマ字欄に残す。
